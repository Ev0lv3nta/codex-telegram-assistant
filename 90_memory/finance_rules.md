# Финансы: правила и категории

Цель: хранить учет просто и стабильно. Лучше 10 понятных категорий, чем 200 запутанных.

## Валюта по умолчанию

- RUB (если пользователь не указал иначе)

## Минимальные категории (пример)

- Еда
- Кафе
- Транспорт
- Дом
- Подписки
- Одежда
- Здоровье
- Подарки
- Образование
- Прочее

## Правила ввода

- Если не указана категория: попытаться угадать по контрагенту/описанию, но пометить как "нужно уточнить".
- Если сумма без валюты: использовать валюту по умолчанию.
- Всегда оставлять короткую заметку, если контекст важен.
