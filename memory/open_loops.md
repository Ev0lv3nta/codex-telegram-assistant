# Открытые линии

Обновлено: 2026-03-07 19:27 (MSK)

- Откалибровать `AUTONOMY_LOOP_POLL_SEC`, `AUTONOMY_DEFAULT_SLEEP_SEC` и `AUTONOMY_BUSY_RETRY_SEC` по реальному поведению уже запущенного state-driven контура.
- Проверить в живом использовании, что idle-контур действительно молчит без пользы и не шлёт пустые техстатусы.
- Проверить в живом использовании, что `DELAY_SEC: 0` даёт короткое продолжение в рамках одного автономного сеанса, а положительный `DELAY_SEC` даёт возврат позже.
- Проверить в живом использовании, что после нового сообщения владельца автономность действительно просыпается по сигналу, а не только по таймеру.
